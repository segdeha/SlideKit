SlideKit was originally conceived by Andrew Hedges and Brian Arnold with help from Jayson Phillips. The current version is based on Andrew's TXJS Roundup.

On the "to do" list:

  • Allowing authors to override the default transition type by putting a classname on the <section> tag
  • Enabling touch/swipe gestures
  • More transitions

Feel free to submit a pull request!

SlideKit is designed to work in Safari or Chrome (or any other browser based on WebKit). It is not an authoring tool. Basically, it's meant for people who know how to edit HTML.

Add a classname to a slide to override the default transition.

<section class="rotate">
  <article><-- content here --></article>
</section>
This is a header

Slides are simple to implement.

<section>
  <aside><-- header here --></aside>
  <article><-- content here --></article>
  <aside><-- footer here --></aside>
</section>
This is a footer
